# WhisMap

Frontend: https://viktorkiramman-arch.github.io/WhisMap/

Firebase Hosting deploy:

1. Set your Firebase project locally:

   ```powershell
   npx firebase-tools use --add
   ```

2. Deploy the static frontend:

   ```powershell
   npm run deploy:firebase
   ```

## Project Commands

Use these commands for this repo:

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Firebase build: `npm run build:firebase`
- Firebase deploy: `npm run deploy:firebase`
- Test: not configured for the frontend
- Lint: `npm run lint`
- Type-check: `npm run check`
- Format: not configured
